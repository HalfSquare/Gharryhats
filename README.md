# CRUDShoppingPortal

This project consists of a web application consisting of a front-end and back-end, and a JSON based REST-ful web service. Both will connect to a MongoDB cluster.


# Basic Requirements
## Web App
- [ ]  User Registration
- [ ] Login / Logout
- [ ] MongoDB as Database
- [ ] View items without Auth
- [ ] Dynamic Endpoints
- [ ] Client Side JS
- [ ] OAuth/OpenID
- [ ] Host on Cloud
- [ ] MVC Architecture

## API
